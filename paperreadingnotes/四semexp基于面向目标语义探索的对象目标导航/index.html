（四）SemExp：基于面向目标语义探索的对象目标导航

Publish on: 2025/12/05
Words: 2050
在未知环境中，依据语义探索导航到目标实例位置。论文发表在2020年的NIPS上，获得了CVPR 2020 Habitat ObjectNac Challenge 的冠军

论文信息

Title: Object Goal Navigation using Goal-Oriented Semantic Exploration

Abstract: This work studies the problem of object goal navigation which involves navigating to an instance of the given object category in unseen environments. End-to-end learning-based navigation methods struggle at this task as they are ineffective at exploration and long-term planning. We propose a modular system called, ‘GoalOriented Semantic Exploration’ which builds an episodic semantic map and uses it to explore the environment efficiently based on the goal object category. Empirical results in visually realistic simulation environments show that the proposed model outperforms a wide range of baselines including end-to-end learning-based methods as well as modular map-based methods and led to the winning entry of the CVPR2020 Habitat ObjectNav Challenge. Ablation analysis indicates that the proposed model learns semantic priors of the relative arrangement of objects in a scene, and uses them to explore efficiently. Domain-agnostic module design allows us to transfer our model to a mobile robot platform and achieve similar performance for object goal navigation in the real-world.

Introduction

自动导航对于构建具身智能至关重要，尤其是在未知环境下自动导航到目标未知，这项任务涉及很多方面，不只是包括目标检测，还包括预测环境与目标之间的相关程度，以做出相对优化的，并且需要对任务有一个长期记忆。 流行的已有做法是使用端到端的强化学习/主动学习，以及通过循环神经网络建立场景记忆以及获取先验语义。 已有研究(Active Neural SLAM)建立了一个障碍地图以维持情景记忆，但是这种方法目标是最大化探索范围，没有对语义编码。 因此作者提出了一种基于语义地图探索的方法，叫做"Goal-Oriented Semantic Exploration(SemExp)"，两个主要贡献如下：

  1. 建立一个增加了语义信息通道的2D俯视地图
  2. 面向目标语义的探索策略，而不是仅通过障碍地图探索

除了赢得了2020年Habitat挑战，，作者还在进行了现实部署

Related Work

Semantic Mapping

基于运动结构恢复和同时定位与建图(SLAM)建立2D/3D障碍物地图已经有很多研究，在地图中融入语义信息已有先例，例如概率图模型或者基于学习的计算机视觉模型，而作者使用可微分(可微数学运算，矩阵变换、插值、加权累积)的投影操作，在地图空间中通过监督来学习语义映射。

概率图模型： 用图来表示随机变量及其依赖关系的概率模型。在语义地图中，将每个栅格看成一个随机变量$X_i$,用一个图把所有格子相连，定义整体概率

基于学习的计算机视觉模型： 深度学习视觉模型(CNN/Transformer等)，进行图像分类、目标检测、语义/实例分割、场景理解等，将语义信息嵌入到地图中。

Navigation

经典的导航方式是使用显式几何地图通过路径规划计算目标位置，目标根据算法进行选择(例如Frontier-based Exploration algorithm)。作者使用基于学习的策略，通过先验语义知识对目标类型进行选择探索。

已有方法：端到端的强化学习/模仿学习

  1. 使用循环神经网络，以及结构化/拓扑空间表示
  2. 在语义信息上构建概率图模型，挖掘环境之间的结构相似性
  3. 使用图卷积神经网络，将先验语义融入到深度强化学习框架
  4. 元强化学习(Meta RL)，智能体学习一种自监督的交互损失
  5. 使用计算机视觉模型，进行语义分割、目标检测，在此之上，使用深度网络训练导航策略

作者指出，显式地图方法要由于纯端到端的学习方法，但是上述方法中的语义信息都是隐式的，作者提出了一种显示的结构化的语义地图表示，使能学习到具有语义感知能力的导航策略

Method

Object Goal Task Definition

对象目标导航任务，给定目标类别 $G$ ，每一个时间步 $t$ ，代理接收视觉观测 $S_t$ 以及位姿传感器信息 $x_t$ ，并执行导航动作 $a_t$ 。 $S_t$包括第一人称RGB图像于深度图像，动作空间$A = { move_ forward, turn_ left, turn_ right, stop }$。当代理认为成功导航到对象目标位置时，即停止，最大时间步为500。

Overview

作者提出了一个名为“Goal-Oriented Semantic Exploration (SemExp) ”的模块化模型，包括两个可学习模块，‘Semantic Mapping’以及’Goal-Oriented Semantic Policy’，前者随着时间构建语义地图，后者做长期策略导航，同时一个’analytical planner’做低层次导航行动控制，系统整体架构如下图所示：

Semantic Map Representation

模型维护一个语义地图$m_t$以及代理的位姿$x_t$，空间地图$m_t$是一个$k * M * M$的矩阵，地图上每个像素点的大小为$25cm^2 (5cm * 5cm)$，M是地图宽度，$K = C+2$是语义地图的通道数量，C是语义种类数目，前两个通道表示空间占用与语义类别。初始时地图初始化为全零，$x_t \in R^3, x_0 = (M/2, M/2, 0.0)$，即初始时，代理总是在地图的中心，并面向东方。

Semantic Mapping

通过已有目标检测预训练模型，进行分类和目标检测建立语义地图，直接根据主动神经SLAM，通过RGB图像生成鸟瞰图。但是鸟瞰图与语义并没有得到融合，作者通过在第一人称视角中预测语义分割，并使用可微投影将第一人称预测转换为自顶向下的映射，但是微小错误经过累积会成为灾难性错误，因此作者通过在地图空间上提出一个损失来克服这个问题。“Overview of the Semantic Mapping module"如下图所示：

基于深度图像计算点云，将点云中每个点与预测类别相关联，点云中的每个点通过几何计算投影到三维空间，得到体素(voxel)表示，再转换为语义地图:对障碍物/栅格/类别，在体素表示的高度维度上进行求和，得到投影语义地图的不同通道，将得到的语义地图输入到去噪神经网络，得到最终的语义地图预测。
