（一）UniGoal：通用零样本目标导航

Publish on: 2025/11/13
Words: 1321
UniGoal: 通用零样本目标导航

一：论文信息

Title: UniGoal: Towards Universal Zero-shot Goal-oriented Navigation

Link: https://arxiv.org/pdf/2503.10630

二：研究背景

Goal-Oriented navigation Classified by Goal Type

  1. Object category: 即Object Goal Navigation

    1. eg：go to the sofa，目标是指定的具体物品的类别

  2. instance image: 即Instance Image Goal Navigaion

    1. eg: 找到图中的沙发，目标是给定图片中物品

  3. text description: 即Text Goal Navigation

    1. eg: go to the chair next to the window，目标是自然语言所描述的物品

三者都是目标导航，但是输入数据的形式不同。其他研究都是针对其中一种进行方法设计，而本文是要提出一个能同时处理上述三个任务的通用架构方法

现存的问题

  1. 现实场景中的人类的指令非常灵活，可以是"类别+图像+自然语言描述"等多种方式出现，因此需要更通用的方法

  2. 方法路线的缺陷

    1. 预训练方法需要在大规模模拟环境中进行预训练，容易对仿真环境过拟合，从而在真实世界中表现不佳

    2. 基于LLM的零样本方法又只针对单一任务进行设计，难以扩展

Frontier-Based Exploration

1. what is Frontier?

在occupancy grid里，每个格子有三种状态（free，occupied，unknow），Frontier是介于unknow与free之间的状态，也就是在frontier这里，你可以看到unknow这个区域的真实状态

2. FBE的工作流程

流程: 利用位姿+RGB-D/激光数据 构建occupied grid，也就是将当前的场景映射为2D的栅格图，检测Frontier，评估每个Fontier并选择其中一个进行探索以更新地图和寻找目标。

规划方法: A*/D* Lite/FMM(Fast Marching Method)

FMM也叫做最快行军法，

相关工作背景

1. Zero-shot Navigation

传统的监督导航方法

Object goal navigation using goal-oriented semantic exploration

Curious representation learning for embodied intelligence

Renderable Neural Radiance Map for Visual Navigation

PONI: Potential Functions for ObjectGoal Navigation with Interaction-free Learning

DD-PPO: Learning Near-Perfect PointGoal Navigators from 2.5 Billion Frames

需要大规模的预训练，泛化能力弱

基于大模型的方法

根据输入的目标类型，分为三类，ON / IIN / TN

ON问题

CoW 基于FBE解决ON问题， ESC ， OpenFMNav ， VLFM 使用LLM提取导航相关尝试辅助探索

IIN问题

Mod-IIN 简单的使用FBE和目标关键点匹配

TN问题

仅有supervised mrthod， Goat and PSL

一句话，就是上面的模型都有缺陷，只针对一种导航子任务进行方法设计，难以支撑真实场景的多模态

关于 Zero-shot

  1. Zero-shot navigation task

    1. 在A环境下训练，在B环境中直接运行，不再对B环境进行建模

  2. Zero-shot category / object

    1. 基于开放词汇识别，测试时出现训练时没有品类

  3. Zero-shot in Function

    1. 不预训练策略，而是直接定义好工作流程

2. Graph-based Scene Exploration

  1. 图表示是一种适合LLM/VLM结合的场景表达方式，其显示编码了物体和关系，便于高层推理

  2. UniGoal集成了三种子任务的导航方法，对于不同的方法，执行不同的处理流程，并且相较于其他方法，有处理上的改进与提升

  • 对 ON：目标图只有一个节点（类别）

  • 对 IIN：图像 → VLM → 物体 + 关系 → 小目标图

  • 对 TN：语言 → LLM 抽取实体 + 关系 → 目标图

三：研究内容

Framework of UniGoal

从图中可以看出，先将三种不同任务类型的输入转化为统一的Goal Graph，同时维持一个在线的场景地图，与在线地图进行匹配，将匹配结果分为三类，零匹配(即未匹配)、部分匹配、完美匹配，根据匹配结果进行探索
