tmux 使用教程（速成版）

tmux 是终端复用器：一个 SSH/终端里开多个窗口与分屏，断开也不丢进程。

0. 安装 

1
2
3
4
5
6

# Debian/Ubuntu/WSL
sudo apt update && sudo apt install tmux
# CentOS/RHEL
sudo yum install tmux
# macOS
brew install tmux

1. 基本概念

  • Session（会话）：最外层容器，可断开/再连。

  • Window（窗口）：会话内的标签页（像 TTY1/TTY2）。

  • Pane（面板）：窗口里的分屏。

默认前缀键是 Ctrl+b（本文记为 <Prefix>）。比如 <Prefix> c 表示先按 Ctrl+b，再按 c。

2. 快速上手（最常用） 

 1
 2
 3
 4
 5
 6
 7
 8
 9

12

tmux new -s work          # 新建会话 work
# 进 tmux 后：
<Prefix> c                # 新建窗口
<Prefix> ,                # 重命名窗口
<Prefix> %                # 垂直分屏
<Prefix> "                # 水平分屏
<Prefix> 方向键           # 在面板间切换
<Prefix> x                # 关闭当前面板（确认）
<Prefix> d                # detach，挂起会话回到普通终端
tmux attach -t work       # 重新连接
tmux ls                   # 列出会话
tmux kill-session -t work # 关闭会话

3. 窗口/面板管理

  • 切换窗口：<Prefix> n（下一个）、<Prefix> p（上一个）、<Prefix> 0..9（按索引）

  • 调整面板大小：<Prefix> Alt+方向键（或 <Prefix> :resize-pane -L 10）

  • 交换面板：<Prefix> { / <Prefix> }

  • 变为单面板：<Prefix> z（缩放/还原当前面板）

  • 同步输入（群发同一命令到所有面板）：

    1
2

    <Prefix> :set -w synchronize-panes on
# 关闭： set -w synchronize-panes off

4. 复制/滚动（Copy Mode）

  • 进入复制模式：<Prefix> [

    • 上下翻：方向键 / PgUp/PgDn / vi 风格（k/j/gg/G）

    • 选取复制（vi 模式）：空格开始、回车复制

    • 粘贴：<Prefix> ]

若要系统剪贴板联动：

  • X11: 安装 xclip 或 xsel，在配置里绑定（见下文）。

  • Wayland: 安装 wl-clipboard。

  • WSL: 安装 clip.exe 相关绑定（见「WSL 贴士」）。

# 更舒服的前缀：Ctrl+a（可选）
set -g prefix C-a
unbind C-b
bind C-a send-prefix

# vi 风格复制 / 更好滚动
setw -g mode-keys vi
set -g mouse on                 # 鼠标支持：点击切 pane/窗口，拖拽调大小

# 更快的切 pane
bind -r h select-pane -L
bind -r j select-pane -D
bind -r k select-pane -U
bind -r l select-pane -R
bind -r H resize-pane -L 5
bind -r J resize-pane -D 5
bind -r K resize-pane -U 5
bind -r L resize-pane -R 5

# 窗口/面板状态更清晰
set -g status-bg colour236
set -g status-fg white
set -g status-interval 2
set -g set-titles on

# 历史滚动更长
set -g history-limit 50000

# 系统剪贴板整合（按需选择其一）
# 1) X11 + xclip:
# bind-key -T copy-mode-vi y send-keys -X copy-pipe-and-cancel "xclip -selection clipboard -in"
# 2) Wayland + wl-copy:
# bind-key -T copy-mode-vi y send-keys -X copy-pipe-and-cancel "wl-copy"
# 3) WSL + clip.exe:
# bind-key -T copy-mode-vi y send-keys -X copy-pipe-and-cancel "clip.exe"

6. 常见工作流模板

一键启动命名会话并预置窗口/分屏 

1
2
3
4
5
6
7

tmux new -d -s dev \; \
  rename-window 'code' \; \
  split-window -h \; \
  send-keys 'htop' C-m \; \
  select-pane -L \; \
  new-window -n 'logs' \; \
  send-keys 'tail -f logs/app.log' C-m \; \
  attach -t dev

在脚本/CI 中运行长任务不掉线 

1
2
3

tmux new -d -s train 'python train.py --cfg cfg.yaml'
tmux ls
tmux attach -t train        # 需要看时再连

7. 会话共享（多人协作）

同一台机器上两个用户/两个终端共享：

1
2
3

tmux new -s pair
# 另一个终端/用户
tmux attach -t pair

不同用户需共享同一 socket 或通过 sudo 调整权限；也可用 tmate 进行安全共享（远程协作）。

8. 记录终端输出（日志） 

1
2

# 将当前面板的历史保存到文件
<Prefix> :capture-pane -S -100000 \; save-buffer output.txt \; delete-buffer

9. 快速速查表（Cheat Sheet）

  • 新会话/接回：tmux new -s NAME；tmux a -t NAME

  • 新窗口：<Prefix> c；重命名：<Prefix> ,

  • 分屏：<Prefix> %（竖） / <Prefix> "（横）

  • 切换面板：<Prefix> 方向键；调整大小：<Prefix> Alt+方向键

  • 缩放面板：<Prefix> z

  • 挂起：<Prefix> d；列会话：tmux ls

  • 复制模式：<Prefix> [；粘贴：<Prefix> ]
