Linux常用命令汇总

Publish on: 2025/10/27
Words: 600
因为总是记不住一些复杂的linux命令，每次使用时总是现场搜索会浪费时间，在此处总结记忆一下

一：远程Linux文件传输

SCP 传输

SCP（Secure Copy）是一种安全的文件传输方法，可以在不同主机之间传输文件。SCP命令通过SSH协议加密传输文件，提高了安全性，但相对速度会慢一些。

1
2
3
4

# 从本地复制文件到远程主机
scp [local directory] [username]@[ip adress]:[remote directory]
# 从远程主机复制文件到本地
scp -rp [ip adress]:[directory] [local adress]

二：cp命令

基本语法

1
2

cp [选项] 源... 目标
cp [选项] 源... 目录/

常用选项

  • -r / -R：递归复制目录（复制文件夹必带）。

  • -a：归档模式 = -dR --preserve=all，最常用于备份/同步（保留权限/时间/属主/软链等）。

  • -p：保留时间戳、权限、属主（--preserve=mode,ownership,timestamps）。

  • -v：展示过程（verbose）。

  • -i：交互式，覆盖前询问（安全）。

  • -n：不覆盖已存在文件（no-clobber）。

  • -u：只在源更新时复制（update）。

  • -f：强制覆盖（force）。

  • -l：创建硬链接而非复制数据（同一文件系统）。

  • -s：创建符号链接而非复制数据。

  • --parents：保留源路径层级结构复制到目标下。

  • -t 目录：把目标目录放前面（便于结合管道或通配符）。

软/硬链接与符号跟随

  • -P：复制符号链接本身（不解引用），等价于 --no-dereference。

  • -L：解引用符号链接，复制链接指向的真实文件。

  • -H：仅对命令行参数中的链接解引用。

归档 -a 默认等价于 -dR，即不解引用符号链接（复制为链接）。
