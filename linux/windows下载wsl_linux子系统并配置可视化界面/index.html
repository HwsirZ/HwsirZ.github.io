Windows下载wsl_linux子系统并配置可视化界面

Publish on: 2025/10/20
Words: 2533
关于如何在windows下安装Linux子系统，并配置图形界面

由于很多项目都是在linux系统下做开发，而目前很多人使用的还都是windows系统的笔记本电脑或者主机电脑，因此windows下的linux子系统是一个不错的选择，但是配置方面仍然会遇到一些问题，本文章主要记录linux子系统的安装以及可视化GUI的配置以及使用

一、安装前的准备

  1. 确认 Windows 版本：WSL 2（推荐版本）需要：

    • Windows 10 2004 及以上版本（内部版本 19041+）

    • Windows 11 任意版本

      检查方法：按下 Win + R，输入 winver 回车，查看 “内部版本号”。

  2. 开启硬件虚拟化：WSL 依赖硬件虚拟化技术（如 Intel VT-x/AMD-V），需在 BIOS 中开启（多数电脑默认开启，若后续安装报错，再按此步骤排查）。

二、安装 WSL（推荐自动安装，10 分钟搞定）

步骤 1：用命令自动安装（最简单）

  1. 以 管理员身份 打开 PowerShell 或 命令提示符（CMD）：

    • 按下 Win 键，搜索 “PowerShell”，右键选择 “以管理员身份运行”。

  2. 输入以下命令，自动启用 WSL 功能并安装默认 Linux 发行版（通常是 Ubuntu）：

    powershell

    1

    wsl --install

    这个命令会自动完成：

    • 启用 “适用于 Linux 的 Windows 子系统” 和 “虚拟机平台” 两个核心功能；
    • 下载并安装 WSL 2 内核；
    • 安装默认的 Linux 发行版（Ubuntu，社区最活跃，适合新手）。

  3. 等待安装完成后，重启电脑（必须重启，否则功能不生效）。

步骤 2：手动选择发行版（可选，若需其他 Linux 系统）

如果想安装 Ubuntu 以外的发行版（如 Debian、Fedora）：

  1. 先执行命令查看可用发行版：

    powershell

    1

    wsl --list --online

    会显示类似列表：

    plaintext

    以下是可安装的有效分发版:
NAME            FRIENDLY NAME
Ubuntu          Ubuntu
Debian          Debian GNU/Linux
kali-linux      Kali Linux Rolling

  2. 安装指定发行版（例如 Debian）：

    powershell

    1

    wsl --install -d Debian

三、初始化 Linux 系统（首次启动）

  1. 重启电脑后，从 “开始菜单” 找到安装的 Linux 发行版（如 “Ubuntu”），点击打开。

  2. 首次启动会自动初始化，等待几分钟（进度条走完）。

  3. 设置 Linux 的 用户名 和 密码（注意：输入密码时屏幕不显示，输完回车即可）：

    bash

    1
2
3

    Enter new UNIX username: yourname  # 输入用户名（如：ubuntu）
New password:  # 输入密码（自定义，如123456）
Retype new password:  # 重复输入密码

    这个用户名和密码是 Linux 系统的管理员（sudo 权限），后续操作需要用。

四、WSL 基础使用（必学操作）

1. 启动与关闭 WSL

  • 启动：

    • 方法 1：从开始菜单点击 Linux 发行版（如 Ubuntu）；
    • 方法 2：在 PowerShell/CMD 中输入 wsl 或发行版名称（如 ubuntu）直接进入。

  • 关闭：

    • 在 Linux 终端中输入 exit 退出当前会话；

    • 若需关闭所有 WSL 实例（如重启服务），在 PowerShell 中执行：

      powershell

      1

      wsl --shutdown

2. 基本 Linux 命令（入门必备）

在 WSL 终端中，可以直接使用 Linux 命令，常用的有：

3. Windows 与 WSL 文件互访（核心优势）

WSL 与 Windows 共享文件系统，无需复杂配置：

  • 在 WSL 中访问 Windows 文件：Windows 的磁盘（C 盘、D 盘）被挂载到 WSL 的 /mnt 目录下，例如：

    • C 盘路径：/mnt/c/
    • 桌面路径：/mnt/c/Users/你的Windows用户名/Desktop/

  • 在 Windows 中访问 WSL 文件：打开文件资源管理器，在地址栏输入 \\wsl$，回车后可看到已安装的 Linux 发行版，双击进入即可访问其文件系统（如 Ubuntu 的文件在 \\wsl$\Ubuntu\home\你的Linux用户名\）。

五、进阶：安装图形界面（可选，如需运行 GUI 程序）

如果需要在 WSL 中运行图形界面程序（如浏览器、Habitat 的可视化工具），需配置 “X11 转发”：

步骤 1：Windows 安装 X11 服务器

下载并安装 VcXsrv（免费开源）：

  • 官网：https://sourceforge.net/projects/vcxsrv/
  • 安装后，启动 “XLaunch”，保持默认配置（“Multiple windows”→“Start no client”→勾选 “Disable access control”），点击 “Finish”（后台运行，任务栏有图标）。

步骤 2：WSL 配置图形转发

在 WSL 终端中执行以下命令（每次启动 WSL 都需要执行，或写入配置文件永久生效）：

bash

1
2

# 设置图形界面转发到Windows的X11服务器
export DISPLAY=$(cat /etc/resolv.conf | grep nameserver | awk '{print $2}'):0.0

步骤 3：测试图形程序

安装一个简单的 GUI 工具（如文本编辑器gedit）测试：

bash

1
2

sudo apt install gedit  # 安装gedit
gedit  # 启动，窗口会在Windows中弹出

笔者安装gedit后发现这个软件的图形化界面只提供了一个基本的文件系统，不能像真正的linux桌面那样。

六：安装xfce4（更完整的图形界面）

第一步：安装 xfce4 桌面所需软件

在 Ubuntu 命令行依次执行命令，搭建桌面环境基础组件：

  • 安装 xfce4 终端（sudo apt-get install xfce4-terminal），为桌面提供终端工具；
  • 安装 xfce4 桌面核心套件（sudo apt-get install xfce4），包含窗口管理器、面板、文件管理器等桌面必需功能；
  • 重启 dbus 服务（sudo service dbus restart），dbus 是 Linux 桌面的消息总线，确保桌面组件间正常通信。

第二步：配置图形转发环境变量（修改～/.bashrc）

通过编辑用户配置文件，让 WSL 每次启动自动设置图形转发相关参数，避免手动重复配置：

  • 用 sudo nano ~/.bashrc 打开配置文件，在末尾添加 3 条环境变量：
    • export DISPLAY=$(grep -m 1 nameserver /etc/resolv.conf | awk '{print $2}'):0.0：自动获取 Windows 的 IP，指定图形界面转发到 Windows 的 X11 服务器（需提前启动 VcXsrv 的 XLaunch）；
    • export WAYLAND_DISPLAY=$DISPLAY 和 export XDG_SESSION_TYPE=x11：强制桌面使用 X11 图形协议（而非 Wayland），避免 WSL 中 Wayland 协议的兼容性问题；
  • 保存退出后，执行 source ~/.bashrc 让配置立即生效，再重启 Ubuntu 确保环境变量稳定加载。

可能上述配置并不能找到正确的ip地址，可通过以下方法设置

1
2
3

ip route show default | awk '/default/ {print $3}'

export DISPLAY=ip地址

上述方式也只是；临时配置，可以通过下面的方式永久配置，在.bashrc文件中永久配置，配置完使用 “source ~/.bashrc”，使配置生效

1
2
3

export DISPLAY=$(ip route show default| awk '/default/ {print $3}'):0.0
export WAYLAND_DISPLAY=$DISPLAY
export XDG_SESSION_TYPE=x11

第三步：启动 xfce4 桌面并在 XLaunch 中显示

配置完成后，在 Ubuntu 命令行执行 xfce4-session 或 startxfce4，此时 Windows 中已启动的 XLaunch 会自动弹出窗口，加载并显示 xfce4 桌面环境，后续可在该窗口中操作桌面（如打开文件管理器、终端等 GUI 工具）。
