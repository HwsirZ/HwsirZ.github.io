多智能体强化学习领域顶级会议总结

1. 领域概述

多智能体强化学习，Multi-Agent Reinforcement Learning，简称 MARL，是强化学习与多智能体系统交叉形成的重要研究方向。其核心问题是：多个智能体如何在共享或竞争环境中学习决策策略，从而实现协作、竞争、通信、协调、任务分配或群体智能。

MARL 论文并不集中在某一个专门会议中，而是主要分布在机器学习、人工智能、多智能体系统、机器人与控制等多个领域的顶级会议中。

2. 最值得关注的核心会议

如果只关注多智能体强化学习领域，最重要的会议主要有以下几个：

会议 类型 关注重点
NeurIPS 机器学习顶会 强化学习算法、MARL 理论、博弈学习、离线 MARL、大规模智能体学习
ICML 机器学习顶会 策略优化、信用分配、强化学习理论、多智能体学习算法
ICLR 深度学习顶会 深度 MARL、表征学习、Transformer-Agent、LLM-Agent、多智能体泛化
AAMAS 多智能体系统顶会 多智能体协作、通信、博弈、机制设计、协调与规划
AAAI 人工智能顶会 多智能体决策、强化学习、规划、博弈与智能系统
IJCAI 人工智能顶会 多智能体系统、博弈论、强化学习、智能决策

其中，NeurIPS、ICML、ICLR 更偏机器学习方法本身，适合关注 MARL 算法创新和理论分析；AAMAS 是多智能体系统领域最对口的会议；AAAI 和 IJCAI 则覆盖人工智能中的多智能体决策、规划、学习和博弈问题。

3. 不同研究方向对应的推荐会议

3.1 偏算法与理论的 MARL

如果研究重点是算法创新、理论分析、样本效率、收敛性、泛化能力、离线强化学习等问题，优先关注：

  • NeurIPS
  • ICML
  • ICLR
  • UAI
  • COLT

这一类会议更看重方法的理论深度、实验充分性和算法普适性。

3.2 偏多智能体系统的 MARL

如果研究重点是智能体之间的协作、通信、竞争、机制设计、博弈均衡、任务分配等问题，优先关注：

  • AAMAS
  • AAAI
  • IJCAI

其中，AAMAS 是最适合多智能体系统方向的会议。如果论文的问题设定非常强调 agent 之间的交互、协调和博弈关系，AAMAS 通常比纯机器学习会议更匹配。

3.3 偏机器人与具身智能的 MARL

如果研究重点是多机器人系统、协作导航、群体机器人、机器人控制、具身智能任务，优先关注：

  • CoRL
  • RSS
  • ICRA
  • IROS

这类会议更重视系统实现、真实机器人实验、仿真到现实迁移以及任务完成效果。对于多机器人协作探索、多智能体目标导航、多机器人路径规划等方向，CoRL、ICRA、IROS 和 RSS 都非常重要。

4. 会议层级与定位

从学术影响力和领域匹配度来看，可以将这些会议大致分为以下几类：

第一类：机器学习顶级会议

  • NeurIPS
  • ICML
  • ICLR

这些会议是强化学习和深度强化学习领域最重要的论文来源。很多高影响力的 MARL 方法，如集中训练分散执行、值函数分解、多智能体策略梯度、离线 MARL、语言智能体协作等，都会出现在这些会议上。

第二类：人工智能综合顶会

  • AAAI
  • IJCAI

这类会议覆盖范围较广，既包括机器学习，也包括搜索、规划、知识表示、博弈论和多智能体系统。适合问题设定更偏 AI 决策系统的 MARL 论文。

第三类：多智能体系统核心会议

  • AAMAS

AAMAS 是多智能体系统领域最核心的会议之一。相比 NeurIPS、ICML、ICLR，AAMAS 更关注智能体之间的交互结构、协作机制、通信机制、博弈关系和社会性行为。

第四类：机器人与控制相关会议

  • CoRL
  • RSS
  • ICRA
  • IROS

这些会议适合多机器人强化学习、协作控制、具身智能、多智能体导航、机器人任务规划等方向。若论文包含真实机器人实验或复杂仿真系统，这些会议通常更合适。

5. 投稿选择建议

不同类型的 MARL 工作适合不同的投稿目标：

研究内容 推荐会议
新的 MARL 算法框架 NeurIPS / ICML / ICLR
多智能体信用分配 NeurIPS / ICML / AAMAS
多智能体通信机制 AAMAS / NeurIPS / ICLR
多智能体博弈学习 AAMAS / AAAI / IJCAI / NeurIPS
离线多智能体强化学习 NeurIPS / ICML / ICLR
多机器人协作导航 CoRL / ICRA / IROS / RSS
群体智能与协作控制 AAMAS / ICRA / IROS
LLM 多智能体协作 ICLR / NeurIPS / AAAI / IJCAI
规划与强化学习结合 ICAPS / AAAI / IJCAI / AAMAS
